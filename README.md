# react-native-console-overlay

A component to display console logs in react native apps.

## Installation

```sh
npm install react-native-console-overlay
```

## Usage

```ts
import { ConsoleOverlayView } from 'react-native-console-overlay';

// ...
<ConsoleOverlayView />;
```
